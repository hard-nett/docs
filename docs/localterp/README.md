# localterp

## Get started

### Clone Osmosis

```
git clone https://github.com/terpnetwork/terp-core.git
cd osmosis
```

### Run localterp

```
make localnet-build
```

### Advanced options

- [Test local Terp-Core core changes](./test-local-changes.md)
- [Run local Terp-Core with a copy of main state](./local-osmosis-mainstate.md)

# CosmWasm - Smart Contracts

## What is cosmwasm?

CosmWasm is a smart contract platform that is supported by Terp Network. In simple terms, it is the Cosmos (Cosm) way of utilizing WebAssembly (Wasm), hence its name.
CosmWasm contracts are genrally written in the software language [Rust](https://www.rust-lang.org/), which is known for its speed & security.
